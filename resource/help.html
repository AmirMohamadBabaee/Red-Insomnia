Name

REDInsomnia - A simple form of Insomnia Application for AP MidTerm Project

Description

REDInsomnia is a personal project for AP MidTerm Project of AUT University. This program powered by jurl command line tool. actually this program convert gui data to this command line tool and show response of this tool in gui form. UI of REDInsomnia designed by swing api in java.

Getting Started

Guide to sending your first HTTP request

To start sending your first HTTP request, you should find your expected URL. This program Only support HTTP and HTTPS protocols. You can enter your expected URL with and without protocol name. If Your entered URL has not any protocol name, REDInsomnia automatically make a HTTP request.
After entering your expected URL, you should select your expected HTTP Method. REDInsomnia support GET, POST, PUT, PATCH and DELETE HTTP methods.
After that, you should add your expected headers. You should add your expected URL as pairs. You can set important headers such as content-type, content-length, connection, vary, accept, keep-alive, cookie, etc. If you Select last header pair, REDInsomnia automatically create new header pair. If you want to remove a specific header pair you can click on trash button in right side of the header pair. Note that You can't remove first item of headers. Each header pair has a checkbox to determine state of the header pair. If checkbox be selected, the pair will be ignored. You can also remove all header pairs with select gear button in left side of the header pair. Note that this button also can not remove first header pair.

Also you can set data in form data part. in this part you can add your expected data such as form data, json and binary file. If you want to send request with form data format, you can use form data panel. this part is like header part. Also, in form data panel you can't remove first form data pair. In this section there is a new button. With this button you can select form of value of form data formation. In normal form, you can't add any text with enter. If you want to add a form value, you can select Multi Line Text. in this file you should add your expected text with enter. if your value has not any enter, REDInsomnia ignore multi line form of this value in later launch. In this situation REDInsomnia automatically add multipart/form-data content-type header.
You can send your expected data in json form too. Note that you can use one of these forms. Either form data or json or binary file. If you want to send JSON data, you should at first right click on form data button then select json. In editor of this page you can add your expected JSON data. REDInsomnia Check if your entered JSON be valid, you can send it else you can't. If your JSON be valid, you can select beautify JSON button, and finally you can send this request. In this situation REDInsomnia automatically add application/json content-type header.
Third form of data transmission is Binary File. To do that, You should right click on json button and select binary file. In this part you can add your expected binary file. In this page, there is a button called reset. you can use this button to reset your selection. Note that you can't add form data, json and binary file simultaneously. you should choose one of them. In this situation REDInsomnia automatically add application/octet-stream content-type header.
If you already save current request, last http request will save in the request. If you want to save initial page you can use save button. In every launch, REDInsomnia go up with initial page that they can be saved.


Options

Follow Redirect

If you enable this option, REDInsomnia automatically follow redirects. When a server can't response to a client, redirect the client to a new url. If this option be disable, REDInsomnia do not follow the redirect and REDInsomnia will show first server response.


Hide On System Tray

In normal situation that this option is disable, when you click close button, REDInsomnia will be terminate. but If you enable this option, REDInsomnia will be hidden in System Tray. That means when you click close button, REDInsomnia will be disappeared but it is still run in background.

Theme

In this section you can select your desired theme. REDInsomnia has two theme : Light(default) , Dark

Note : When you confirm your setting, REDInsomnia will be restarted and unsaved data will be removed. so if you have necessary data in initial page or in saved request, that have not saved, Don't Confirm Setting. at first, save your data and request and then change REDInsomnia Setting.



View
